# 🏥 مركز الجلدية والليزر - نظام إدارة متكامل PWA

نظام إدارة متكامل ومتطور لمراكز الجلدية والليزر، يعمل كتطبيق ويب تقدمي (PWA) مع دعم كامل للعمل بدون إنترنت.

## ✨ المميزات الرئيسية

### 🏠 لوحة التحكم (المركز)
- إحصائيات شاملة (المرضى، الزيارات، الإيرادات، المصروفات)
- عرض زيارات اليوم
- المواعيد القادمة مع إمكانية الإرسال عبر واتساب

### 👨‍💼 قسم المساعد
- تسجيل المرضى الجدد
- أنواع الزيارات (كشف، إعادة، جلسة، كشف + جلسة)
- إدارة الجلسات والأسعار
- سجل الحضور اليومي
- إدارة الحجوزات والمواعيد
- سجل المكالمات
- إدارة التحاليل والفحوصات
- سجل المرضى الكامل مع البحث

### 👨‍⚕️ قسم الأطباء (محمي بكلمة سر)
- إدارة الأطباء والنسب
- بحث المرضى
- معرض صور قبل/بعد مع ضغط تلقائي
- قاعدة بيانات الأدوية
- الروشتات الجاهزة (نسخ سريع)
- حساب نسب الأطباء

### ✨ قسم الليزر
- تسجيل جلسات الليزر
- إعدادات الليزر (Fluence, Pulse Duration, Spot Size)
- مناطق العلاج والأسعار
- باقات الليزر
- متابعة نتائج العلاج

### 📁 قسم الملفات (محمي بكلمة سر)
- إدارة ملفات المرضى
- ضغط الصور تلقائياً
- التاريخ الطبي لكل مريض

### 🔍 البحث المتقدم
- بحث بالاسم والهاتف والعنوان
- فلترة بالتاريخ
- فلترة بنوع الزيارة

### 📊 التقارير (محمي بكلمة سر)
- التقارير المالية (يومي، أسبوعي، شهري)
- تقارير الزيارات
- تقارير الجلسات
- سجل المصروفات
- رسوم بيانية تفاعلية

### ⚙️ الإعدادات (محمي بكلمة سر)
- **12 لون ثيم**: أزرق، أخضر، بنفسجي، أحمر، برتقالي، تيل، وردي، سماوي، نيلي، عنبري، زمردي، روز
- **الوضع الليلي**: تصميم مريح للعين
- **نظام الحماية**: كلمة سر للأقسام الحساسة
- **المزامنة P2P**: بين الأجهزة على نفس الشبكة
- **النسخ الاحتياطي**: يدوي وتلقائي
- **حذف البيانات**: كلي أو جزئي

## 🔒 الحماية

- كلمة السر الافتراضية: `2137`
- يمكن تغييرها من الإعدادات
- الأقسام المحمية: الأطباء، الملفات، التقارير، الإعدادات

## 📱 PWA Features

- يعمل بدون إنترنت (Offline)
- قابل للتثبيت على الهاتف
- إشعارات Push
- مزامنة في الخلفية

## 🚀 التثبيت والاستخدام

### الطريقة 1: فتح مباشر
1. افتح `index.html` في المتصفح
2. سيطلب المتصفح تثبيت التطبيق

### الطريقة 2: خادم محلي
```bash
# باستخدام Python
python -m http.server 8000

# أو باستخدام Node.js
npx serve .

# ثم افتح http://localhost:8000
```

### الطريقة 3: GitHub Pages
1. ارفع الملفات إلى مستودع GitHub
2. فعّل GitHub Pages من Settings
3. افتح الرابط

### الطريقة 4: أي استضافة ويب
- ارفع الملفات الثلاثة إلى أي استضافة

## 💾 قاعدة البيانات

- يستخدم IndexedDB للتخزين المحلي
- جميع البيانات محفوظة على جهازك
- لا حاجة لسيرفر خارجي

## 🔄 المزامنة

- يستخدم BroadcastChannel API
- يعمل بين تبويبات المتصفح المختلفة
- يمكن تفعيله من الإعدادات

## 📤 النسخ الاحتياطي

- **يدوي**: من الإعدادات > نسخ احتياطي
- **استعادة**: رفع ملف JSON
- ملف النسخة بصيغة JSON قابل للقراءة

## 📱 واتساب

- إرسال تذكيرات للمواعيد
- التواصل مع المرضى
- رسائل جاهزة مسبقاً

## 🎨 التخصيص

- 12 لون مختلف
- وضع ليلي/نهاري
- واجهة عربية RTL

## 📊 المتطلبات التقنية

- متصفح حديث (Chrome, Firefox, Safari, Edge)
- لا يحتاج تثبيت أي برامج
